# ⏱️ Stopwatch Web Application

A simple and responsive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. The application allows users to accurately measure elapsed time with features such as Start, Pause, Reset, and Lap Recording.

## 🚀 Features

- ▶️ Start the stopwatch
- ⏸️ Pause the stopwatch
- 🔄 Reset the timer
- 🏁 Record lap times
- 📱 Responsive and user-friendly interface
- ⚡ Real-time time tracking with millisecond precision

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
